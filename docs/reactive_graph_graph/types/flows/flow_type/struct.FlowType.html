Expand description

Flow types defines the type of an flow instance like a template for flow instances.

They contain entity instances and relation instances. The wrapper entity instance is mandatory and used for input and outputs.

Fields§

§ty: FlowTypeId

The type definition of the entity type.

§description: String

Textual description of the flow type.

§wrapper_entity_instance: EntityInstance

The wrapper entity instance.

§entity_instances: EntityInstances

The entity instances which are contained in this flow.

By default, no entity instances are contained in this flow type.

§relation_instances: RelationInstances

The relation instances which are contained in this flow.

By default, no relation instances are contained in this flow type.

§variables: Variables

The variables. Variables will be replaced by instantiation of a flow instance.

By default, the flow type has no variables.

§extensions: Extensions

Flow type specific extensions.

On the builder, call .ty(...), .description(...)(optional), .wrapper_entity_instance(...), .entity_instances(...)(optional), .relation_instances(...)(optional), .variables(...)(optional), .extensions(...)(optional) to set the values of the fields.
