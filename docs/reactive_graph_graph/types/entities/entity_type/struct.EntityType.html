Expand description

Entity types defines the type of an entity instance.

Fields§

§ty: EntityTypeId

The type definition contains the namespace and the type name.

§description: String

Textual description of the entity type.

§components: ComponentTypeIds

The names of the components of the entity type.

§properties: PropertyTypes

The properties which are defined by the entity type.

§extensions: Extensions

Entity type specific extensions.

pub fn builder() -> EntityTypeBuilder<((), (), (), (), ())>

Create a builder for building EntityType. On the builder, call .ty(...), .description(...)(optional), .components(...)(optional), .properties(...)(optional), .extensions(...)(optional) to set the values of the fields. Finally, call .build() to create the instance of EntityType.
